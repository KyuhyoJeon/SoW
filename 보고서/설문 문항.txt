독서량과 방식 및 시각장애를 위한 독서도우미 프로그램 선호조사

1. 시력으로 문자를 읽는데 불편함을 갖고 있거나 주변에 갖는 사람이 있습니까? ( O / X )
     
    1-1. (1번 문항에 O를 했다면 답변) 불편을 갖고 있는 사람과의 관계는? ( 본인 / 가족이나 가까운 지인 / 아는사람 )

2. 점자 형태의 문자를 많이 보셨나요? ( O / X )

    2-1. (2번 문항에 O를 했다면 답변) 주로 어디에서 보셨나요? ( 공공장소의 안내 / 점자책 / 기타 (                                 ) )

    2-2. (2번 문항에 O를 했다면 답변) 점자 형태로 문자를 많이 읽나요? ( O / X )

        2-2-1. (2-2번 문항에 X를 했다면 답변) 읽지 않는 이유는 무엇인가요? ( 점자를 읽을줄 몰라서 / 읽을줄 알지만 어렵고 힘들어서 )

3. 평소에 독서를 얼마나 하시나요? ( 독서는 안한다 / 책은 좋아하지만 잘 안한다 / 책은 싫은데 업무상 독서를 한다 / 시간 날 때마다 하는 편이다 )

    3-1. (3번 문항에 '독서를 안한다', '책은 좋아하지만 잘 안한다'를 했다면 답변) 책을 읽지 않는 이유는 무엇입니까? ( 글을 읽는게 귀찮고, 힘들어서 / 시간이 없어서 / 독서 말고 다른것을 하고싶어서 )

    3-2. (3번 문항에 '책은 싫은데 업무상 독서를 한다'를 했다면 답변) 책이 싫은 이유는 무엇입니까? ( 글을 읽는게 귀찮고, 힘들어서 / 시간이 없어서 / 독서 말고 다른것을 하고싶어서 )

4. 문서나 책을 촬영하면 글을 읽어주는 프로그램이나 제품에 대해 보거나 듣거나 사용해 본 적이 있나요? ( O / X )

    4-1. (4번 문항에 O를 했다면 답변) 해당 프로그램이나 제품에 대해 설명해주세요

    4-2. (4번 문항에 O를 했다면 답변) 해당 프로그램이나 제품의 편한 점, 불편한 점에 대해 작성해 주세요

5. 그런 프로그램이나 제품이 있다면 사용할 용의가 있나요? ( O / X )

6. 그런 프로그램이나 제품이 시력에 불편함을 가진 사람에게 도움을 줄 수 있다고 생각하시나요? ( O / X )

7. 그 밖에 시각장애인들에게 도움이 될 프로그램이나 제품에 대해 생각해본 것이 있나요?
